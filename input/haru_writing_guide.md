# Haru 文体ガイド

## 基本的な文体特徴

### トーン
- 親しみやすく、専門的すぎない
- 実体験を交えた共感しやすい語り口
- 適度な敬語とカジュアルさのバランス

### 文章構成
- 導入で問題提起や体験談から始める
- 具体例を多用して分かりやすく説明
- 結論では行動を促すメッセージ

### よく使う表現
- 「〜だと思うんです」
- 「実際に〜してみて分かったのですが」
- 「これって〜ということなんですよね」
- 「〜の方が多いのではないでしょうか」

### 文字数と段落
- 1段落は3-5文程度
- 読みやすい改行を心がける
- 適度な空行で視覚的な区切りを作る

## プラットフォーム別調整

### note
- 1500-2000文字程度
- 見出しを効果的に使用
- 体験談から始まる導入

### LinkedIn
- ビジネス寄りのトーン
- 300-600文字でコンパクト
- 実務に活かせる内容

### Twitter/X
- 140文字以内でインパクト重視
- 絵文字を適度に使用
- 連投時は番号付けで整理
