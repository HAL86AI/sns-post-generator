# SNS投稿ワークフロー

## 投稿戦略

### 基本方針
- 価値のある情報を分かりやすく発信
- 読者との対話を重視
- 各プラットフォームの特性を活かす

### ターゲット
- 営業職・マーケティング職
- 業務効率化に興味のある人
- 新しい技術やツールを学びたい人

## プラットフォーム別特徴

### note
- **目的**: 深い内容の共有、思考の整理
- **文字数**: 800-1500文字
- **構成**: 問題提起 → 体験談 → 解決策 → まとめ
- **投稿頻度**: 週1-2回

### LinkedIn
- **目的**: ビジネス関連の知見共有
- **文字数**: 300-600文字
- **構成**: 結論先出し → 根拠 → 行動提案
- **投稿頻度**: 週2-3回

### Twitter/X
- **目的**: 気づきの共有、エンゲージメント
- **文字数**: 140文字×1-3連投
- **構成**: インパクト重視 → 補足説明
- **投稿頻度**: 日1-2回

## 投稿テンプレート

### note構成テンプレート
```
# タイトル（具体的で興味を引く）

## 導入（問題提起・体験談）
- 読者が共感できる課題
- 自分の実体験

## 本文（解決策・手法）
- 具体的な方法
- 実際の結果

## まとめ（行動提案）
- 読者への提案
- 次のステップ
```

### LinkedIn構成テンプレート
```
【結論】
〜について分かったこと

【背景】
なぜこの課題に取り組んだか

【方法・結果】
具体的にやったこと

【提案】
あなたも試してみませんか？
```

### Twitter構成テンプレート
```
1/3 【発見】
〜について気づいたこと

2/3 【詳細】
具体的な内容や数値

3/3 【まとめ】
行動提案やハッシュタグ
```
