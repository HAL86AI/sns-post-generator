# 【事務職のキャリア】「AIに仕事、奪われますか？」と聞かれたら、私がいつも答えていること。

こんにちは、合同会社アークスのはるです。

AIの話題が当たり前になった今、セミナーや勉強会で、決まっていただく質問があります。

「AIがもっと賢くなったら、私たちの仕事はなくなってしまうんでしょうか？」

その不安な気持ち、とてもよく分かります。20年間、事務という仕事に誇りを持って向き合ってきた私にとっても、他人事ではありません。

そんな時、私はいつもこうお答えしています。

「AIに奪われる仕事は、確かにあるかもしれません。でも、それは私たちが『本当にやるべき仕事』に集中するための、素晴らしいチャンスなんです」と。

## AIが得意なこと、人がやるべきこと

私がAIと一緒に働いていて感じるのは、彼らが「正確な作業」や「膨大なデータ処理」のプロフェッショナルだということです。

一方で、AIにはできないこともたくさんあります。

例えば、こんな場面。

*   困っている同僚の気持ちを察して「大丈夫？」と声をかけること。
*   お客様の言葉の裏にある、本当のニーズを汲み取ること。
*   チームの雰囲気を和ませる、ちょっとした雑談をすること。

これらはすべて、血の通った人間にしかできない、とても価値のある「仕事」です。

AIの登場は、私たち事務職が長年担ってきた「作業」の部分を、少しずつ手放す機会をくれているのだと思います。そして、その代わりに「人との繋がり」や「思いやり」といった、人間ならではの強みにもっと時間を使えるようになる。私はそう信じています。

## 「AIに仕事をさせる」人になる

これからの事務職に求められるのは、AIに仕事を「奪われる」のではなく、AIに仕事を「させる」スキルではないでしょうか。

*   「このデータ分析、お願いね」とAIに指示を出すスキル。
*   AIが出してきた答えを鵜呑みにせず、最終判断をするスキル。
*   AIによって生まれた時間で、新しい価値を生み出すスキル。

AIを「仕事を奪うライバル」ではなく、「業務を助けてくれるアシスタント」として捉え直す。その視点の転換こそが、AI時代を生き抜く鍵だと感じています。

## まとめ：事務職の未来は、もっと面白くなる

20年前、手書きだった伝票がExcelに変わった時も、きっと同じような不安があったはずです。でも、私たちはExcelを使いこなし、もっと創造的な仕事を生み出してきました。

AIとの関係も、きっと同じです。

AIに単純作業を任せ、私たち人間はもっと人間らしい仕事に集中する。そんな未来を想像すると、私はとてもワクワクします。

事務職の未来は、決して暗いものではありません。むしろ、もっと面白くなる。私は、心からそう信じています。

最後まで読んでいただき、ありがとうございました！
